Enumeration

ModalPresentationStyleCompat

Defines the modal presentation style of the View when compiled for iOS 13 or iOS 14.

Declaration

enum ModalPresentationStyleCompat

Topics

Enumeration Cases

case automatic
The default presentation style chosen by the system.
case custom(transitioningDelegate: UIViewControllerTransitioningDelegate?)
A custom view presentation style that is managed by a custom presentation controller and one or more custom animator objects.
case formSheet(properties: SheetPropertiesCompat?)
A presentation style that displays the content centered in the screen.
case overFullScreen
A view presentation style in which the presented view covers the screen.
case pageSheet(properties: SheetPropertiesCompat?)
A presentation style that partially covers the underlying content.
case popover(permittedArrowDirections: UIPopoverArrowDirection, sourceRectTransform: ((CGRect) -> CGRect)?, adaptiveSheetProperties: SheetPropertiesCompat?)
A presentation style where the content is displayed in a popover view.
